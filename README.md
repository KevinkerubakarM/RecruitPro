# RecruitPro

Hire top talent 10x faster with AI-powered candidate matching. 95% match accuracy. 50K+ successful hires.

# Technology

Recruit Pro is built on NextJS, which helps to build seamless frontend and backend applications with SEO optimized webpages
Integrated with Cloudinary to store Banners,Logo,Culture Videos.
Prisma Postgres is used for storing all user profile,jobs,career page information.

# Setup Guide

1. Download and install Node version 25.x.x from https://nodejs.org/en/download
2. Clone the git repository https://github.com/KevinkerubakarM/RecruitPro.git to your local system.
3. Open terminal and navigate to RecruitPro directory
4. Verify node and package manager installation by running command `node -v` then `npm -v`
5. Run command `npm i` to install project dependencies
6. For development, run command `npm run dev` to run web application locally
7. Open browser url http://localhost:3000 to open web application
8. For production, first run command `npm run build` to build the project files and then run command `npm run start` to start web application

Login Page - Recruiter or Candidate can enter their email,password to enter their respective home page.
Recruiter will redirected to Recruiter home page
Candidate will redirected to Candidate home page
Recruiter Home -
Create Career page and Job posting option - Recruiter first creates a company career page then can create and link job posting to that company career page.

Recruiter Company Career Builder Page -
This page allows Recruiter to design their company career page,can upload banner,logo,culture video,about us, meet our team and custom selection
Recruiter can build multiple company career page but not same company.
